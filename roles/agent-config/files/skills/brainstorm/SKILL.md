# Brainstorm

When the path isn't obvious, don't jump to the first idea.

1. **Frame** — restate the goal and hard constraints in one line. List what's still unknown or assumed.
2. **Diverge** — generate 3–5 *genuinely distinct* approaches (not variations of one). For each: the idea in a sentence, why it fits, its main tradeoff, and rough effort (S/M/L).
3. **Stress-test** — call out risks, edge cases, and the assumptions that would need validating first.
4. **Converge** — recommend one option with a one-line rationale, and note the single fact that would flip the decision.

Keep it a tight decision aid, not an essay. Don't start building until the direction is chosen.
